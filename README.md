# CestaCompra
Actividad 1: Carrito de la compra agoodshop

Objetivos de la actividad

En esta actividad debes programar la funcionalidad de carrito de la compra de una tienda online ficticia. Aquí se pretende poner en práctica:

▸	Llamadas a una API
▸	Manejo y modificación del DOM
▸	Programación orientada a objetos

Aunque la tienda es ficticia, la manera de enfocar esta actividad y lo que se requiere no difiere mucho a cómo funciona el mundo real.
